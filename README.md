# Realized-Volatility-Forecasting-Comparison
A replication and benchmark of the Heterogeneous Autoregressive model of Realized Volatility (HAR-RV) from Corsi (2009), evaluated against a Random Forest and a shallow MLP on S&amp;P 500 realized volatility. Performance is assessed using QLIKE loss, Mincer-Zarnowitz regressions, and Diebold-Mariano tests against a naïve persistence baseline.
